                  등록된 학사 일정

                  이번 달 등록된 학사 일정이 없습니다.

                  48차시 마스터 교구 정보 관리

                  1년 동안 사용할 48차시의 표준 교구 리스트입니다. 필요시 항목을 더블클릭하여 수정할 수 있습니다.

                  학교별 월간 교구 스케줄러

                  각 학교의 실제 수업일(1~5주차)에 진행할 교구를 자유롭게 재배치합니다.

                  💡 공휴일 휴강으로 진도가 밀릴 때 조율 팁

                  예컨대 5월에 대체공휴일로 2번만 수업하여 교구가 밀린 경우:

                  • 1. 6월 1주차에 밀린 5월 드론 교구(예: 12차시)를 지정합니다.
                  • 2. 6월 실제 수업 주차(예: 3주)에 맞추어 6월용 교구를 순서대로 지정합니다.
                  • 3. 이번 달에 쓰지 못하고 남게 된 나머지 6월 교구는 슬롯 우측의 [보관] 버튼을 클릭하여 창고(보관고)로 즉시 이체한 뒤 다음 달 계획 시 당겨 쓸 수 있습니다.

                  남은 교구 보관고 (창고 인벤토리)

                  휴강, 공휴일, 학생 결석 등으로 밀려 보관 중인 교구들의 수량과 히스토리를 관리합니다.

                  대시보드 학사일정 스케줄러 보관고

                  ☁️ 클라우드 동기화 설정

                  📌 Firebase 무료 설정 방법 (5분)
                  1. console.firebase.google.com → Google 로그인
                  2. 프로젝트 추가 → 이름 입력 → 계속 → 프로젝트 만들기
                  3. 왼쪽 메뉴 → 빌드 → Realtime Database → 데이터베이스 만들기
                  4. 위치: 기본값 → 테스트 모드로 시작 → 사용 설정
                  5. 표시되는 URL 복사 (https://xxxxx-rtdb.firebaseio.com)
                  6. 아래에 붙여넣기 + 동기화 키 설정 → 연결하기
                  🗂 오프라인 백업 (선택사항)

                  Firebase 연결 없이도 JSON 파일로 데이터를 저장/복원할 수 있습니다.
